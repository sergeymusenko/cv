Content Management Systems

2001-2002.

   Articles And News Input Interface - DocBuilder.
Creates an ordered structure of articles which may be used by other programs as web-site main menu, news, articles, pages of sites etc.

Visual intuitively understandable interface allows managing a web-site operatively and making changes of contents in wide range. Any user who is not familiar with HTML may form pages containing paragraphs, lists, web-links, tables and images, apply style definitions, attach binary files (for example MS Office files, PDFs, archives etc) to page.
Besides DocBuilder sets additional attributes to articles, that allows to automate change of contents of a web-site in dependences on registered user or time of viewing.
To each article it is possible to add "discussion" -- flame of user messages moderated by web-site manager.
There is a system of differentiation of access levels for different logins of web-site managers.

Author's project.
Implementation CKEditor, PHP, HTML, JavaScript, SQL Sybase, SQL MySQL.

Internet-programming

November 2005

   SMS-chat, operator and admin interfaces
Providing of a dialogue between mobile phone abonents and operators of a chat.
Project consists of 2 parts:
• SMS Sending/Receiving Server;
• Operator/Dispatcher Web-interface.

Last one consists of 3 parts:
• Operator Interface (dialogue with abonents, incoming messages queue management, operator's online status control) - it should be many operators online simultaneously [pic.1];
• Dispatcher Interface (operator's accounts management, information flow control, stats and reports) [pic.2];
• Master Administrator Interface (operator's and dispatcher's accounts management, system management and settings, data backup service) [pic.3].

A complicated logic of incoming messages queue has been realized. Criterions are: dialogue pre-history between each abonent and all operators, operator's online status and size of incoming messages queue and dispatcher's manual redirections of incoming messages.

Author's project.
Implementation PHP, HTML, JavaScript, MySQL
