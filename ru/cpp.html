Май-август 2002 г.

   Школьное расписание School
Программа для автоматизации составления расписания занятий средней школы.

Визуальный интерфейс расстановки уроков с подсветкой нежелательных и запрещенных ячеек. Вывод на печать (по листам) готового расписания.

Авторский проект.
Реализация: Borland C++ Builder 5.0 (BDE, Paradox).

Август 2001 г.

   DataWall ограничение доступа к web-данным
Часть проекта "AmCham Ukraine".

FCGI-приложение под Linux для обеспечения авторизованного доступа к файлам данных сайта с одновременным кэшированием.

Авторский проект.
Реализация: С++ (gcc).

Июль-август 2001 г.

   Демон чата chatd
Часть проекта Wap-Chat.

Участие в разработке программы обслуживания клиентов чата. Обеспечивает регистрацию пользователя, создание/удаление комнат, прием/рассылку сообщений, сохранение/восстановление данных на диск.
Работает в среде Unix/Linux.

Реализация: С++ (gcc).

Август-сентябрь 2000 г.

   Donor - регистратура
Система автоматизации регистратуры Областной станции переливания крови, г. Сумы.

Ведение учета сдачи крови (более 30 тыс. записей), поиск донора по нескольким критериям, автоматическая проверка невхождения в "черный список" или контактирования с лицами из "черного списка", печать направлений и справок, архивирование данных.

Авторский проект.
Реализация: Borland C++ Builder 5.0 (BDE, Paradox).

Февраль 1994 г.

   BBASE - библиографическая база данных

Небольшое Windows-приложение для обработки библиографических данных (просмотр, сортировка, поиск, фильтр, печать), хранящихся в dbf-файлах.
Была представлена фирмой "Максима" на выставке Комтек на Красной Пресне в Москве в марте 1994 года.

Авторский проект.
Реализация: Borland Turbo C 3.0 с использованием OWL и CodeBase.

1991-2000 гг.

   KASHA - пищеблок больничного стационара

Программа для автоматизации работ по ведению учета и оценке экономической эффективности деятельности пищеблока и продуктового склада больничного стационара.

Позволяет получать следующие документы и отчеты:

  • меню-требование;
  • меню для отделения;
  • требование на буфет;
  • оборотная ведомость;
  • расходная накладная;
  • стоимость дня;
  • карточка-раскладка;
  • список продуктов на складе;
  • натуральные нормы расхода продуктов;
  • коечный фонд;
  • коечный фонд по данным буфета;
  • ведомость расчетов с поставщиками.
Программа работает под MS-DOS / Windows9x и эксплуатируется по сей день в трех больницах города Сумы, Украина.

Авторский проект.
Реализация: Clipper 5.2
