Визуальный интуитивно понятный интерфейс позволяет администратору сайта произвольно и оперативно менять содержимое сайта. С его помощью пользователь, не знакомый с HTML, может формировать текст, содержащий параграфы, списки, линки, таблицы и изображения с определением стилей, а так же добавлять связанные с текстом двоичные файлы, такие, как файлы MS Office, архивы и т.п.
Кроме того, программа задает дополнительные признаки статьи, что позволяет автоматизировать изменение содержания сайта в зависимости от категории зарегистрированного пользователя или времени просмотра.
К каждой статье возможно добавить "обсуждение" -- пользовательские сообщения, модерируемые администратором сайта.
Имеется система разграничения доступа администраторов сайта к каждому из разделов структуры статей.

Авторский проект.
Реализация: CKEditor, PHP, HTML, JavaScript, SQL Sybase/MySQL

Комплекс управления SMS-чатом

Суть проекта: организация SMS-диалогов между абонентами мобильной связи и операторами чата.
Проект состоит из двух частей:
• сервер приема и отправки сообщений и
• комплекс управления SMS-чатом.

Последний в свою очередь состоит из трех частей:
• интерфейса оператора (ведение диалогов с абонентами, автоматическое уведомление о поступившем сообщении, контроль присутствия оператора в онлайне; операторов в системе может быть несколько) [pic.1];
• интерфейса диспетчера (контроль загрузки операторов, управление аккаунтами операторов, сбор статистики) [pic.2];
• интерфейса главного администратора (управление аккаунтами операторов и диспетчеров, общая настройка функционирования системы, резервное копирование данных) [pic.3].

Особенность проекта заключается в механизме определения оператора, которому передается вновь поступившее сообщение. Учитывается предыстория диалогов абонента со всеми операторами, присутствие в онлайне и загрузка оператора, а так же действия диспетчера по перенаправлению абонентов.

Авторский проект.
Реализация: PHP, HTML, JavaScript, MySQL

Система "Недвижимость Киева".

Система предназначена для автоматизации работы агента по недвижимости.
Включает три составляющие:
  • скрипт периодического сбора информации;
  • скрипт размещения информации в базе данных;
  • web-интерфейс пользователя для осуществления выборки информации.
Первая часть отвечает за сбор актуальной информации с электронных досок объявлений и сайтов рекламных изданий.
Вторая часть распознает тексты, выделяя из них ключевые параметры, по которым третья часть производит выборки, и помещает их в базу данных.
Третья часть, в свою очередь, позволяет сформировать условия выборки информации по более, чем 16 параметрам.

Использование системы обеспечивает высокий уровень интерактивности в работе с клиентами и позволяет агенту сэкономить несколько часов рабочего времени ежедневно.

Авторский проект.
Реализация: Python, PHP, MySQL

Система загрузки рингтонов на мобильные телефоны.

Система выбора и загрузки рингтонов на мобильные телефоны абонентов оператора мобильной связи "КиевСтар", Украина.

Программа позволяет при помощи SMS интерактивно выбрать требуемую композицию и отправить ее на мобильный телефон, используя специальный интерфейс оператора мобильной связи (CPI - Content Provider Interface).
Поддерживается несколько видов запросов:

  • запрос общей справки;
  • запрос списка поддерживаемых моделей телефонов;
  • навигация по многоуровневому каталогу композиций;
  • поиск композиции по строке;
  • выбор композиции в проводимых акциях (хитпарад, новый год и т.п. - короткий код композиции).
Для формирования каталога композиций создан специальный web-интерфейс.

Авторский проект.
Реализация: Python, SQL Sybase
